About

Welcome to The Korean Handbook where you will get to learn the basic Korean language known as Hangul or 한굴. This website is developed and maintained by Daniel Hearn but is open source on Github if you would like to contribute content (I would especially appreciate audio).

But enough with the introduction, lets get started!
